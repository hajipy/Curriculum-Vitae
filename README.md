## 個人活動

### 個人開発アプリ Frames

2017年末からElectron + Vue.js + TypeScriptで「Frames」という名前のアプリを個人で開発しています。

アプリの概要は開発2周年に書いた以下の記事をご参照ください。  
[https://hajipy.net/2019/12/individual-dev-2019/](https://hajipy.net/2019/12/individual-dev-2019/)

上記記事を書いたあとも開発は継続しており、アプリとして公開するための体裁が整いました。

- アプリアイコン
    - 作成スキルがないため、クラウドワークスで発注
- 利用許諾同意画面
- リリースノート画面
- 自動バージョンアップの仕組み
- バージョンアップ時のデータマイグレーションの仕組み
- GitHub Actionsを利用したCI/CD環境の構築

現在は友人数十名に協力してもらい、クローズドベータテストをしています。  
実行しているPCのローカル環境にファイルをたくさん作るアプリであるため、不具合がないか不安でしたが、テスターの皆さんからは問題なく動いているとのフィードバックをいただいています。  
ニッチなアプリなのですが、数名のテスターからは「こういうアプリが欲しかった！」とのフィードバックをいただいており、開発を続けるモチベーションとなっています。
